# 教程

欢迎来到 Tuack-NG 的使用教程！在这里，你将学会安装 Tuack-NG，创建你的第一个工程，编写题面，渲染题目 PDF，并为其添加测试用例与标准解法（Std），使用 Std 与数据生成器生成数据，将工程导出到评测平台以进行测试，或生成下发文件以提供给选手。

## Tuack-NG 是什么？

Tuack-NG 是一套辅助 OI/ICPC 出题的工具包，它涵盖了以下方面：

- 生成题面 PDF / Markdown
- 编写测试用例进行测试
- 使用 Std 和数据生成器生成数据
- 导出工程到 OJ 或 评测机

同时，它也是 [Tuack](https://gitee.com/mulab/oi_tools/) 的继任者。
